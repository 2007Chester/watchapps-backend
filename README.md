# WatchApps Backend — Магазин циферблатов и приложений для Wear OS

WatchApps — это платформа для покупки, загрузки и установки циферблатов и приложений для Wear OS.  
Проект включает backend (Laravel 11), Android приложение, Wear OS клиент, Dev Console и сайт.

Backend предназначен для:
- авторизации пользователей и разработчиков
- загрузки APK
- выдачи файлов Android-приложению
- управления каталогом циферблатов
- учёта покупок и платежей
- взаимодействия Android ↔ Wear OS
- синхронизации установки APK на часы

---

## 🚀 Стек технологий

- PHP 8.3  
- Laravel 11  
- MariaDB  
- Nginx  
- Laravel Sanctum  
- REST API (JSON)  
- File Storage / Uploads  
- HTTPS / SSL
