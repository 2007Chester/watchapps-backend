# WatchApps Dev Console

Панель разработчика для платформы **WatchApps**.

## Назначение

- Регистрация и управление аккаунтом разработчика
- Загрузка APK циферблатов и приложений
- Управление товарами (названия, описания, цены, категории)
- Управление медиа (иконки, баннеры, скриншоты, GIF)
- Просмотр статистики продаж
- Просмотр выплат

## Технологии (план)

- Web‑фронтенд (React / Next.js)
- Взаимодействие с backend API WatchApps
