# Архитектура WatchApps Dev Console

## Основные разделы UI

- Dashboard (обзор продаж и товаров)
- Мои циферблаты
- Создать новый циферблат
- Статистика
- Финансы
- Настройки аккаунта

## Взаимодействие с backend

- Авторизация разработчика
- CRUD операций по watchfaces
- Загрузка файлов (через /api/upload)
- Статистика продаж и выплат
